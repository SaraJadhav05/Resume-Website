# Resume-Website

Resume Website Enhancement

Objective:

To enhance the design and layout of a personal resume website using CSS styling 
concepts including fonts, color schemes, positioning, and selectors.

Improvements Made:

1. Navigation Bar with Section Links:
Added a responsive and fixed navigation bar using Flexbox that includes links to 
major sections of the resume like Education, Skills, Projects, etc. Clicking these links 
smoothly scrolls to the respective sections, improving user navigation.

2. External CSS File:
Used an external stylesheet (style.css) to separate structure from styling. This 
ensures cleaner HTML and makes the site easier to maintain and update.

3. Professional Font Integration:
Imported and used the Google Font 'Montserrat' to enhance text readability and give 
the resume a modern and professional look.

4. Profile Image Integration:
Added a profile picture (profile.jpg) at the top section to personalise the resume and 
make the presentation more engaging.

5. Section Layout and IDs:
Structured the resume into clear sections using semantic HTML5 tags (<section>,
<header>, etc.) with unique IDs for internal linking and accessibility.

7. Color Theme & Styling:
Applied a cohesive, professional color scheme with soft highlights, interactive hover 
effects, shadowed cards, and rounded corners. These design choices enhance the 
visual appeal while maintaining a professional tone.

8. Responsive Design:
Ensured that the layout is responsive across screen sizes using flex and percentage-
based widths, along with media queries (if required later).

9. Hover & Transition Effects:
Added smooth transitions and hover effects to navigation links and content blocks to 
make the user experience more dynamic and interactive.

10. Clean Alignment and Spacing:
Used margin and padding strategically to give content enough breathing room. 
Ensured the alignment of headings, lists, and cards for clarity and visual balance.
